Cursos disponibles


BackEnd

Los cursos de backend son programas de aprendizaje diseñados para enseñar a los estudiantes cómo desarrollar y construir la parte del servidor de una aplicación web. Estos cursos suelen estar dirigidos a personas interesadas en convertirse en desarrolladores web completos, ya que el conocimiento del backend complementa el desarrollo frontend (lado del cliente).


Java

Java es un lenguaje de programación versátil, orientado a objetos y multiplataforma, usado ampliamente en desarrollo web y aplicaciones.


Sql Server

SQL Server es un sistema de gestión de bases de datos relacional de Microsoft, ampliamente utilizado en empresas y aplicaciones empresariales.


Python

Python es un lenguaje de programación versátil, legible y de fácil aprendizaje, utilizado en desarrollo web, ciencia de datos y automatización.


PHP

PHP es un lenguaje de programación de código abierto y ampliamente utilizado para el desarrollo web dinámico y la creación de aplicaciones web.


FrontEnd

Los cursos de frontend son programas educativos que se enfocan en enseñar a los estudiantes cómo desarrollar la parte visual y de interacción de una aplicación web. El frontend es la cara visible de la aplicación, con la que los usuarios interactúan directamente a través de sus navegadores. Estos cursos son ideales para aquellos interesados en convertirse en desarrolladores web con habilidades específicas para crear interfaces atractivas y funcionales.


HTML

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para crear la estructura y contenido de páginas web.


CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño que controla el aspecto y presentación visual de páginas web HTML.


JavaScript

JavaScript es un lenguaje de programación de alto nivel, utilizado para agregar interactividad y dinamismo a páginas web y aplicaciones.


Angular

Angular es un framework de desarrollo web de código abierto de Google, basado en TypeScript, para crear aplicaciones web SPA (Single Page Applications).
